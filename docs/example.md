# ZecHub Example Doc

ZecHub is a community-driven wiki and resource hub for Zcash.

## What is Zcash?

Zcash (ZEC) is a privacy-preserving cryptocurrency that uses zk-SNARKs to allow users to transact privately.
Unlike Bitcoin, Zcash supports shielded transactions that hide the sender, receiver, and amount.

## Shielded vs Transparent Addresses

- **Transparent (t-addr):** Visible on the blockchain like Bitcoin.
- **Shielded (z-addr):** Fully private using zk-SNARKs.

Always prefer shielded addresses for maximum privacy.

## Getting Started

1. Download a Zcash wallet such as Zashi or Nighthawk.
2. Generate a shielded address.
3. Obtain ZEC from an exchange.
4. Send a shielded transaction to your z-addr.
